É necessário executar primeiro os scripts do postgres e depois o script the instalação install.bat, dado que são feitas migrações para o postgres dentro desse script.


PgAdmin:

Para começar no pgadmin, estamos a usar a versão postgres 15. O primeiro script a executar é o script "criar_utilizador", de seguida "criar_base_dados" ->  "sequencias" -> "tabelas_scripts" -> "popular_base_dados" -> "criar_vistas" -> depois desses a ordem é indiferente, mas todos os scripts têm de ser executados.


MongoDB:

No MongoDB, o nome da databse tem de ser 'trabalho_final', vão existir 3 coleções, "carrinhos", "produtos" e "users". Para que seja possível utilizar a plataforma web é necessário criar os utilizadores para que possa ser feito o login. O ficheiro a inserir na coleção "users" para que isso seja possível é "trabalho_final_users.json". De seguida é necessário importar o ficheiro "produtos_mongo.json" para popular a coleção produtos.
